# WeChat-GroupExporter
Author : slingthy
Dependencies : python-docx, lxml

提取微信群单人的所有文本类的聊天记录
操作流程：登陆Windows微信客户端，在微信群里无脑手动拉，复制所有的聊天记录
创建Word文档，粘贴，保存为“xxx.docx”
最终文档和原始文档位于同一目录
Example:
path : C:*\*\*
filename ：**1.docx
who : ID
save as : **2.docx
